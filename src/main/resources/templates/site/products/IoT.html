기술 제품 소개

압전 세라믹/폴리머 복합 소재 기반의 플렉서블 센서를 이용한
"미소풍력을 이용한 독립전원형 IoT 델리네이터"

차량이 이동할 때 발생하는 바람으로부터 LED 발광 기능을 갖는 델리네이터

개발 제품 구성도

분야별

기 술

  • 미소풍력 에너지 변환 기능을 갖는 고내구성
    플렉서블 에너지 변환 소자

  • 저전력 컨트롤 유닛 및 저전력 IoT 무선 센서 통신 모듈

  • 독립전원 모듈 기반 델리네이터 일체화
    App/Web 기반 센서 데이터 모니터링 프로그램

  • 스마트 IoT 델리네이터 기반 환경
    (온도, 미세먼지, NOx) 모니터링 시스템

제 품

  • 자가발전IoT 기능의델리네이터 제품

  • LED + 발전용 압전소자 + 배터리+ 통신모듈 등으로 구성

  • 종래 제품보다 고기능, 고가격

  • 지자체나 한국도로공사 측에 납품

기술측면

미소풍력을 전기로 변환하는 에너지 변환 소자기술로 야간에만 활용되는
델리네이터의 단순기능을 초월한 에너지 자립형 IoE (에너지 인터넷)
혁신기술로 안전, 환경, 통신분야의 다목적 다기능 역할 가능

  • 기술의 특장점

    필름형 압전소자,
    우수한 전기적 &
    기계적 압전성능
    (유연성, 내구성, 내충격성,
    성형가공 편리)

    날씨(안개, 연무),
    비산먼지 영향을
    받지 않음
    (위험도로에서의 안전
    효율 2.5배 증가)

    간단한 플래팅 방식의
    Simple, Smart
    (도서산간 화재,
    재난, 안전경보)

    웨어러블 압전센서
    +
    IoT(Internet of Thing)
    =
    미활용 에너지를 활용

  • 기술의 기대효과

    에너지 자립화 기능을
    같은 원천기술확보

    C-ITS(쌍방향ITS)
    M2M매체 역할 수행

    IoE 기술을 활용한
    안전 시장의
    다목적, 다기능

    스마트 IoT
    +
    에너지 자립형 IoT
    =
    신시장 개척
